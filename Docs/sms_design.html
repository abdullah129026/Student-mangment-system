03   Class Responsibilities
Class File Pair Responsibility
Student Student.h / .cpp Holds student personal data. CRUD on individual student.
SchoolClass Class.h / .cpp Manages multiple classes (1-A, 2-B…), lists students per class.
Result Result.h / .cpp Stores marks per subject, calculates total, percentage, grade.
Attendance Attendance.h / .cpp Tracks present/absent days, computes attendance percentage.
Fees Fees.h / .cpp Records fees paid/pending, handles payment transactions.
FileManager FileManager.h / .cpp All binary file I/O: read/write/update/delete records from .dat files.
ReportGenerator ReportGenerator.h / .cpp Generates formatted .txt reports for result, attendance, fees, full.
